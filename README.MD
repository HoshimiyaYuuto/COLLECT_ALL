## 遊戲說明
### 很久很久以前，有一個村子正逢百年一度的大饑荒
### 村子內存在個古老傳說: 若饑荒之日降臨，將必有一位勇者被天神施予加持，前往收集仙食之征途以終結饑荒
### 善良又強大的你就這樣被抬舉成勇者。你背負著全村的希望，踏上收集仙食的道路
### 臨行前，村子裡的老祖宗遞給你仙食尋覓指引:
1. 每處關卡您需收集所有仙食，全部收集完(24種)才算過關
2. 有些仙食會閃爍光芒、縮放大小、上下跳動甚至有意識地移動，切勿為此驚慌
3. 每處關卡存在2~5位仙食守護者，必須不被他們抓到，否則任務失敗
4. 進去每處關卡後，必須在 3 分鐘內完成收集，不然會觸動關卡的警報機制使得任務失敗
5. 有點累了可以偷偷暫停時間，站在原地不動休息一下，仙食守護者們不會趁這時抓人
6. 關卡存在BGM和你生成/摧毀技能方塊的聲音都不是幻聽，雖然他們不會招來仙食守護者(也許)，不過如果你不喜歡可以禁音它們
7. 老祖宗也有付給你技能方塊的屬性互剋指引，如果你忘記能開啟 Property Rules.MD 看看
### 祝你好運，勇者

## Execution & Deployment Guide

### Option I (Recommended): Use run.bat
* **Requirements**: The host machine must have a Java 21 (JDK/JRE) runtime environment.
* **How to Run**: In a Windows environment, simply double-click **`run.bat`** in the root directory to launch.

### Option II: Use an IDE
* **Requirements**: The host machine must have Java 21 (JDK/JRE) and related JavaFX library runtime environments.
* **How to Run**: Import this project into your preferred Java IDE (e.g., IntelliJ IDEA) and run `src/main/java/program/intro_to_cs_lab_final_project/Main.java`.

---

## Controls
1. Use WASD to move up, left, down, and right.
2. Use Space to spawn or destroy character skill blocks.

---

## Game Lore & Instructions
### Once upon a time, there was a village suffering from a once-in-a-century great famine.
### An ancient legend existed in the village: when the day of famine arrives, a hero chosen and blessed by the heavens will embark on a journey to collect divine food to end the famine.
### Kind and powerful as you are, you were chosen as the hero. Carrying the hopes of the entire village, you set forth on the path to gather the divine food.
### Before your departure, the village elder handed you the "Divine Food Seeking Guide":
1. In each stage, you must collect all divine food. The stage is cleared only after gathering all 24 types.
2. Some divine food may flash, resize, bounce up and down, or even move consciously. Do not panic.
3. There are 2 to 5 Divine Food Guardians in each stage. You must avoid being caught by them, otherwise, the mission fails.
4. Once entering a stage, you must complete the collection within 3 minutes; otherwise, the stage's alarm system will trigger, leading to mission failure.
5. If you get tired, you can secretly pause time to rest by standing still. The Divine Food Guardians will not catch you during this time.
6. The stage BGM and the sound of spawning/destroying skill blocks are not hallucinations. Although they won't attract Guardians (probably), you can mute them if you prefer.
7. The elder also attached an attribute counter guide for the skill blocks. If you forget, you can check `Property Rules.MD`.
### Good luck, Hero.

## 実行・デプロイ方法

### 選択肢 I (推奨)：run.bat の使用

* **起動方法**：お使いの Java IDE（IntelliJ IDEA など）に本プロジェクトをインポートし、`src/main/java/program/intro_to_cs_lab_final_project/Main.java` を実行してください。

## ゲーム説明
### 遠い昔、ある村が百年に一度の大飢饉に見舞われていました。
### 村には古くから伝わる伝説がありました。「飢饉の日が訪れし時、天の加護を受けし勇者が現れ、大飢饉を終わらせるために仙食を集める旅に出る」と。
### 善良で力強いあなたは、こうして勇者に祭り上げられました。村全体の希望を背負い、あなたは仙食を集める旅へと一歩を踏み出します。
### 出発の直前、村の長老から「仙食探索の手引き」を手渡されました：
1. 各ステージのすべての仙食を集める必要があります。全24種類をすべて集めるとステージクリアとなります。
2. 一部の仙食は光ったり、大きさが変わったり、上下に跳ねたり、さらには自らの意思で動いたりしますが、決して慌てないでください。
3. 各ステージには2〜5人の「仙食の守護者」がいます。彼らに捕まらないようにしてください。捕まると任務失敗になります。
4. ステージに入った後、3分以内に収集を完了しなければなりません。時間を過ぎると警報が鳴り、任務失敗となります。
5. 疲れた時は、その場に立ち止まってこっそり時間を一時停止し、休憩することができます。この間、守護者たちが襲ってくることはありません。
6. ステージのBGMや、スキルブロックを生成・破壊する時の音は幻聴ではありません。守護者を引き寄せることはありません（おそらく）。もし気になるならミュートにすることも可能です。
7. 長老はスキルブロックの属性相性表も持たせてくれました。もし忘れてしまったら、`Property Rules.MD` を開いて確認してください。
### 勇者の幸運を祈る。
